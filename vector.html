Vector Processing and Analysis

Importing vector data

GRASS supports a multitude of vector data formats. Raster data formats supported by OGR can be easily be imported into a GRASS mapset. 

v.in.ogr
Vector outputs from GRASS GIS can be exported to OGR supported formats easily. 

v.out.ogr

Vector Modules and Processes

Raster modules and processes begin with v.*. Most of them can be found in the Vector dropdown menu.
For more information, visit:
  • GRASS GIS Manual: Vector data processing in GRASS GIS
  • GRASS GIS Manual: vector
